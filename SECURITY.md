# Security Policy

## Supported Versions

We actively support and apply security patches to the following versions of Beloved:

| Version | Supported |
| ------- | --------- |
| v0.1.x  | ✅ Yes     |
| < v0.1  | ❌ No      |

## Reporting a Vulnerability

We take the security of the Beloved application assembly engine seriously. If you find a security vulnerability, please report it responsibly.

**Do NOT open a public GitHub issue for security bugs.**

Please report vulnerabilities by emailing security@beloved.build. 

Include the following information in your report:
- A description of the vulnerability and its potential impact.
- Detailed steps to reproduce the issue (and proof-of-concept scripts/configurations if available).
- The version of the Assembly Engine, Control Plane, or CLI where the issue was observed.

We will acknowledge receipt of your report within 48 hours and coordinate a disclosure timeline.
